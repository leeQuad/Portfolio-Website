Portfolio Website

A personal portfolio website built using HTML, CSS, and JavaScript to showcase my projects and technical skills. The website serves as a central place for employers and recruiters to learn more about my experience and ongoing development in IT and cybersecurity. As this was my first web development project in a while, I had to spend time refreshing my knowledge of HTML, CSS, and JavaScript. One of the biggest challenges was ensuring the website remained responsive across different screen sizes, particularly when designing the mobile navigation menu and project card layouts. While there are still improvements I would like to make, I am pleased with the final result and believe it is a strong first project.
