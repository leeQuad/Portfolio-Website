Personal Portfolio Website

Overview

This is my first project which is a responsive personal portfolio websitethat is designed to showcase my skills, projects and professional background. The website serves as a central hub where employers and recruiters can learn more about my experience in IT, cybersecurity, networking, and web development.

Features

Fully responsive design for desktop, tablet, and mobile devices Modern user interface with smooth navigation Contact form for professional enquiries Downloadable CV option Clean and accessible design links to socials a Project details Technologies Used

HTML CSS JavaScript Git GitHub Pages Challenges As this was my first web development project in a while, I had to spend time relearning concepts and solving issues as they came up. The biggest challenge was making the website look good on both desktop and mobile devices, especially the navigation menu and project cards. Although there are still improvements I would like to make, I am proud of the outcome and feel it is a solid first project.

What I Learned

Through this project I improved my understanding of responsive web design, CSS layouts, version control using Git, and deploying websites through GitHub Pages. I also gained experience troubleshooting layout and compatibility issues across devices.
