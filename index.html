Featured Project

AI Resume Analyzer

Built a Python and Streamlit application that compares an uploaded resume against a pasted job description, extracting key skills from both and calculating a match score to show how well the resume fits the role.

  • Python
  • Streamlit
  • PDF Parsing
  • Skill Extraction

  1. Problem

    Manually checking whether a resume matches a job description is slow, and it's easy to miss skills that an ATS would screen for.

  2. Solution

    Built a Streamlit app that accepts an uploaded resume PDF and a pasted job description, then extracts and compares the skills found in each.

  3. Outcome

    Returns a list of detected skills and an ATS match score as a percentage, giving instant feedback on how closely a resume matches a role.

Key Features

  • Uploads a resume as a PDF and accepts a pasted job description
  • Extracts and lists detected skills from the resume
  • Calculates and displays an ATS match score as a percentage

What I learned Built my first interactive web app with Streamlit, worked with PDF text extraction and skill/keyword matching, and structured a multi-file Python project — separate modules for parsing, skill extraction, and scoring — instead of one script.

  • File Integrity Monitor

    Developed a script to get a hands-on understanding of how defensive security tools catch unauthorised system changes. It hashes a set of files, stores the results, and flags any file that changes so unauthorised edits are easy to spot.

    • Python
    • SHA-256
    • File Integrity Monitoring
    • Hashing

    1. Problem

      Unauthorised file changes on a system can go unnoticed until it's too late.

    2. Solution

      Built a tool that checks files and compares their current hashes against previously stored ones. If a file has been added, modified, or deleted, the program displays a notification showing the change.

    3. Outcome

      Flags any file that's changed, been added, or gone missing, so tampering is easy to catch.

    Key Features

    • Generates SHA-256 hashes for monitored files
    • Detects changed, new, and deleted files on re-scan
    • Stores file hashes in the database

    What I learned By building this tool, I improved my Python file handling and JSON skills while learning how to use SHA-256 hashing to verify file integrity.
